  <files>

    src/content/portfolio.ts

  <action>
    For each of the 4 source images provided by the user:

    1. RESIZE to 1280px wide using sips:
       `sips --resampleWidth 1280 {source} --out /tmp/{name}-resized.png`

    2. CROP to exactly 1280x720 (16:9). Use sips cropToHeightWidth to crop from top:
       - First check height: `sips --getProperty pixelHeight /tmp/{name}-resized.png`
       - If height > 720, crop to 720 keeping the top (hero area): `sips --cropToHeightWidth 720 1280 /tmp/{name}-resized.png --out /tmp/{name}-cropped.png`
       - If height < 720, pad or scale-fit (unlikely at 1280w for a website screenshot)

    3. CONVERT to WebP at quality 80 using cwebp:
       `cwebp -q 80 /tmp/{name}-cropped.png -o public/images/portfolio/{output-name}.webp`

    Output file names (must match existing paths in portfolio.ts):
    - `liat-leshem-before.webp`
    - `liat-leshem-after.webp`
    - `bialystok-before.webp`
    - `bialystok-after.webp`

    4. VERIFY each output image is exactly 1280x720:
       `sips --getProperty pixelWidth --getProperty pixelHeight public/images/portfolio/{name}.webp`
